# 概要
架空のピザレストランのホームページです。

# 開発環境・使用言語
MAMP
<br>
HTML ,Sass(SCSS) ,Javascript(jQuery)

# 制作について
gulpとbrowserifyの導入から行いました。

FLOCSS設計に基づいて制作しました。

またコンテンツ表示などにCSSアニメーションを使うことで、最後まで閲覧したくなるようなデザインを意識しました。
